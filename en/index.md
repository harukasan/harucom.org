Harucom is a handmade single-board computer you can get for about $20.
You can connect a keyboard and a TV directly, so you can experience Ruby programming without using a PC.

The name "Harucom" carries several meanings:

- **H**euristically **A**ccessible **Ru**by **Com**puter
- a **Ha**ndmade **Ru**by **Com**puter
- **Haru**kasan's Ruby **Com**puter

Harucom aims to be an accessible computer where you can learn programming intuitively.
By removing the complexities of modern computing, the goal is to let you touch the essence of programming.
